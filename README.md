# Browser Launcher Pro

A powerful browser extension for managing multiple browser versions and enhancing your web browsing experience.

## Features

- **Multi-Browser Management**: Launch and manage different versions of Chrome and Edge browsers
- **Version Tracking**: Monitor browser versions and receive update notifications
- **WSL Integration**: Seamlessly work with Windows Subsystem for Linux
- **Quick Search**: Search across multiple engines (YouTube, Google, DuckDuckGo, etc.)
- **Customizable UI**: Personalize the extension's appearance with custom colors
- **Keyboard Shortcuts**: Quick access to common functions
- **Incognito Mode Support**: Launch browsers in private browsing mode

## Installation

1. Download the extension from the Chrome Web Store
2. Click "Add to Chrome" to install
3. Accept the permissions when prompted

## Usage

### Browser Management
- Click the extension icon to open the popup interface
- Select your desired browser version to launch
- Enable version tracking to receive update notifications

### Search Features
- Select text on any webpage
- Right-click to access search options
- Use keyboard shortcuts (Ctrl+Y for YouTube, Ctrl+Shift+Z for incognito)

### WSL Integration
- Enable WSL features in settings
- Configure your preferred WSL instance
- Launch terminal sessions directly from the extension

## Settings

- **Appearance**: Customize background and font colors
- **Search Engines**: Enable/disable specific search providers
- **Notifications**: Configure update notification preferences
- **WSL**: Manage WSL instance settings
